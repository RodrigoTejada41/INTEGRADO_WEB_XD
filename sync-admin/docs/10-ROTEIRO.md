# Roteiro

## Description
Página legada de navegação para a evolução planejada do `sync-admin`.

## Structure
- [`modules/README.md`](./modules/README.md)
- [`CHANGELOG.md`](../CHANGELOG.md)
- [`docs/12-DOSSIE-STATUS-ATUAL.md`](./12-DOSSIE-STATUS-ATUAL.md)

## Integrations
- Segurança
- Monitoramento
- Auditoria

## Flow
1. Consulte o dossiê para o estado atual.
2. Consulte o changelog para a evolução.
3. Use esta página como atalho de navegação.

## Critical Points
- O roteiro não deve competir com o dossiê.
- A visão de futuro deve permanecer resumida.

## Tests
- Conferência de consistência com o changelog e o dossiê.
