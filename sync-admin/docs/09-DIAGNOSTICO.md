# Diagnóstico

## Description
Página legada de navegação para incidentes, causas e correções do `sync-admin`.

## Structure
- [`modules/README.md`](./modules/README.md)
- [`modules/services.md`](./modules/services.md)
- [`modules/web-ui.md`](./modules/web-ui.md)

## Integrations
- Logs
- Dashboard
- Changelog

## Flow
1. Considere o módulo de serviços para a causa raiz.
2. Use esta página como índice de navegação.

## Critical Points
- Documentar a correção em um único local.
- Evitar repetir as mesmas anotações em vários lugares.

## Tests
- Reproduzir, corrigir e confirmar ausência de regressão.
