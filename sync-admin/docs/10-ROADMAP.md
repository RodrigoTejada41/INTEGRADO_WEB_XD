# Roadmap

## Fase 1 (atual)
- Ingestao via API com autenticacao por chave.
- Persistencia de lotes e registros.
- Painel administrativo com dashboard e historico.

## Fase 2
- Multiempresa com isolamento por tenant e usuario.
- Multiunidade por filial/terminal com politicas de acesso.
- Exportacao Excel/PDF.

## Fase 3
- Integracao com Obsidian/Nexus para trilha de auditoria.
- Alertas automatizados.
- Escalabilidade horizontal da API.
- Rate limiting e controles avancados de seguranca.
