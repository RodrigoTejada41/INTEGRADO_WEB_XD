# Release Notes - v0.1.0

## Summary
Initial production-ready foundation for the synchronized data platform.

## Highlights
- Modular API and agent architecture.
- Incremental sync with `uuid`-based upsert.
- Multi-tenant validation with `empresa_id`.
- Retention policy limited to 14 months.
- Admin panel with login, dashboard, records, history, and settings.
- Tenant provisioning and key rotation from the panel.
- Health checks, metrics, and automated tests.

## Validation
- Automated suite: `8 passed`
- Release tag: `v0.1.0`

## Notes
- This release establishes the base for the next commercial phase.
- The upcoming line of work is focused on deeper multi-company isolation, richer exports, and advanced monitoring.
