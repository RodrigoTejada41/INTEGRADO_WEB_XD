# CEREBRO_VIVO
Base local de conhecimento para este projeto.

## Ordem de consulta
1. Leia esta base local em `README.md` e `00_PAINEL/INICIO.md`.
2. Consulte `07_INDICES/indice-geral.md` e `Conhecimento/hubs/index.md`.
3. Revise `00_PAINEL/PROCESSO_PROJETO.md` e `Logs/processo_projeto.json` para retomada.
4. Use a web apenas quando a base local e os índices não responderem.

## Uso
- Atualize a vault local antes de buscar fora do projeto.
- Registre novas notas, conversas e aprendizados aqui primeiro.
- Se este workspace estiver conectado ao CLI do CEREBRO_VIVO, a configuração local fica em `config/indexer_config.json`.
